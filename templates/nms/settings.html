봇 설정 가이드

Telegram 봇 만들기

  1. 텔레그램에서 @BotFather 검색하여 대화 시작
  2. /newbot 명령 입력
  3. 봇 이름 입력 (예: NMS Bitic Bot)
  4. 봇 username 입력 (예: nms_bitic_bot)
  5. 발급된 Bot Token 복사
  6. 봇을 그룹에 초대하거나 1:1 대화 시작
  7. Chat ID 확인: https://api.telegram.org/bot{TOKEN}/getUpdates 에서 chat.id 값 복사
  8. 또는 @userinfobot 에게 메시지 보내서 ID 확인

Discord Webhook 만들기

  1. Discord 서버에서 서버 설정 클릭
  2. 연동 (Integrations) 메뉴 선택
  3. 웹후크 (Webhooks) 클릭
  4. 새 웹후크 만들기 클릭
  5. 알림 받을 채널 선택
  6. 웹후크 URL 복사
  7. 복사한 URL을 위 채널 추가에 입력
